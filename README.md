# PyMerger

**Объединение Python-файлов в один TXT для загрузки в AI-чаты**

## 📌 Описание

PyMerger — это утилита с графическим интерфейсом для объединения нескольких файлов `.py` в один TXT-документ.

**Зачем это нужно?**
- AI-чаты (ChatGPT, Claude, DeepSeek) ограничивают загрузку файлов (обычно до 5 штук)
- PyMerger объединяет любое количество файлов в один
- Вы можете загрузить весь проект целиком!

## ✨ Возможности

- ✅ Объединение любого количества `.py`-файлов
- ✅ Сохранение структуры с заголовками (`=== имя_файла.py ===`)
- ✅ Поддержка UTF-8 (русские символы, комментарии)
- ✅ Простой графический интерфейс на tkinter
- ✅ Удаление файлов из списка
- ✅ Автоматическое открытие папки с результатом
- ✅ Добавление даты и количества файлов в результат
